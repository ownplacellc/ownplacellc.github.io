Премиум-голос: бесплатный ключ Azure

Премиум-голос Дмитрий в ListenBook работает через ваш собственный бесплатный ключ Microsoft Azure (тариф Free F0: ~500 000 знаков в месяц — примерно один роман). Ключ хранится только на вашем устройстве (в Keychain) и никуда не передаётся, кроме запросов синтеза к Azure. Озвученные главы остаются в библиотеке навсегда и слушаются офлайн.

Понадобится: email, около 10 минут. Microsoft может попросить банковскую карту для подтверждения личности — деньги при этом не списываются, а тариф Free F0 бесплатный без срока действия.

Шаг 1. Создайте бесплатный аккаунт Azure

Откройте azure.microsoft.com/free, нажмите Try Azure for free и войдите с любым аккаунтом Microsoft (или создайте новый), следуя подсказкам регистрации. Новым аккаунтам Microsoft также даёт $200 кредита на 30 дней — для нашей задачи он не нужен: тариф Free F0 бесплатен сам по себе.

Шаг 2. Создайте ресурс «Speech service» (тариф Free F0)

Важно: создавайте именно классический ресурс «Speech service». Не создавайте ресурс «AI Foundry» — у Foundry нет бесплатного тарифа, там платный S0 с первого запроса.

Быстрый путь — прямая ссылка на форму создания. Или вручную: portal.azure.com → Create a resource → в поиске наберите Speech service.

В форме создания:

  • Resource group — «Create new», любое имя (например, listenbook);
  • Region — ближайший к вам (например, East US). Запомните его — он понадобится в приложении;
  • Name — любое свободное имя;
  • Pricing tier — обязательно Free F0.

Нажмите Review + create → Create и дождитесь «Your deployment is complete».

Шаг 3. Скопируйте Key 1 и Region

Нажмите Go to resource, слева в разделе Resource Management откройте Keys and Endpoint. Скопируйте Key 1 (кнопка копирования справа от поля) и запомните Location/Region (например, eastus).

Никому не передавайте ключ. Если ключ «засветился» — на той же странице есть кнопка Regenerate.

Шаг 4. Вставьте ключ в ListenBook

В приложении: Настройки → Облачный голос (бета) → вставьте ключ и регион → Проверить и сохранить. Появится зелёное «Ключ работает» — готово: при импорте текстовой книги выбирайте голос «Дмитрий (облако)».

Если что-то не работает

СообщениеПричина и решение
Неверный ключ или регионКлюч скопирован не полностью, или регион не совпадает с регионом ресурса. Регион пишется слитно строчными: eastus, не «East US».
Месячный бесплатный лимит исчерпанТариф F0 даёт ~500 000 знаков в месяц. Лимит обнулится в новом месяце, либо продолжите бесплатным локальным голосом.
Слишком много запросовНа бесплатном тарифе — не более 20 запросов в минуту. Приложение подождёт и продолжит само; это нормально для длинных книг.
Ошибка соединенияПроверьте интернет и правильность региона.

Инструкция проверена: июль 2026. Лимиты тарифа Azure может менять Microsoft — актуальные значения: страница цен Azure Speech.
